# Homebrew Tap

Custom Homebrew tap for TimBroddin's projects.

## Install

```bash
brew tap TimBroddin/tap
brew install --cask divoomalbumart
brew install --cask gitstatus
```

## Available Casks

| Cask | Description |
|------|-------------|
| `divoomalbumart` | Display album art on Divoom LED panels |
| `gitstatus` | macOS menubar app that monitors git status of local repositories |
